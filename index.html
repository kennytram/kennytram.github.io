MockCord

Live Site | GitHub

MockCord is a web-based communication platform that enables users to engage in real-time chat messaging. The application provides a seamless and user-friendly interface, allowing individuals and groups to communicate effectively and efficiently.

This project was developed using a tech stack consisting of React-Redux for the front-end, Ruby on Rails for the back-end, and a PostgreSQL database.

Pixel Panda

Live Site | GitHub

Pixel Panda is an e-commerce web application designed for game artists to showcase and sell their artwork assets. The application is built using the MERN stack, which includes MongoDB, Express.js, React.js, and Node.js.

I worked in collaboration with two other engineers, contributing to both the front-end and back-end of the application. My primary responsibility was to ensure seamless communication between the two sides, guaranteeing that data was effectively transmitted to where it was needed.

Mayhem Overload

Live Site | GitHub

Mayhem Overload is a game inspired by Warioware, developed entirely in pure Javascript with asynchronous programming. The game features fast-paced challenges that test players' reflexes and agility, with a variety of mini-games and levels to keep things interesting.
